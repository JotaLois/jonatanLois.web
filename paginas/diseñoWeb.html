DISEÑO WEB

Diseño UX - UI de la app "getLabApp", Aplicación destinada a la búsqueda laboral. La misma se realizó como proyecto central del curso de Diseño UX - UI de Coderhouse. Para su realización se tuvieron en cuenta metodologías ágiles. Design thinking. Card sortings y esquemas de flujo. Confección de proto personas y user personas. Prototipado en diferentes niveles de calidad hasta llegar al prototipo final.

Sitio web del Arquitecto Guillermo Papini. Arquitecto graduado de la Universidad de Buenos Aires

Sitio web de la start up Incubadora de tesis. Proyecto realizado a partir del curso de desarrollo web de Coderhouse.
